# Setup

## Requisitos
- [Docker for Windows](https://docs.docker.com/docker-for-windows/install/) rodando.
- Visual Studio 2017.
- .Net Core 2.0.

## Rodando
- Abra o projeto usando a solução **WappaMobile.sln**
- Compile a solução
- Marque **docker-compose** como projeto default
- Execute (F5)
- Abra um navegador e acesse `http://localhost:5000/`
